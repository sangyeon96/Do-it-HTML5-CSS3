### 01-3 HTML 기본 문서 구조

* 태그, 이건 꼭 알아두세요!
* HTML 문서 기본 구조 살펴보기
* &lt;!doctype&gt; - 문서 유형을 지정하는 선언문
* &lt;html&gt; 태그 - 웹 문서 시작을 알리는 태그
* &lt;head&gt; 태그 - 브라우저에게 정보를 주는 태그
* &lt;meta&gt; 태그 - 문자 인코딩 및 문서 키워드, 요약 정보
* &lt;body&gt; 태그 - 실제 브라우저에 표시될 내용
* 웹 문서에서 특수 문자 및 특수 기호 사용하기
